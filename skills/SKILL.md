# PaperSkills

Use this skill as the entry point for academic research and paper-writing tasks.

## When To Use

Use PaperSkills when the user asks to:

- Search or map literature
- Identify research gaps
- Verify citations or supporting evidence
- Draft or revise abstracts
- Perform peer review
- Match a manuscript to journals
- Generate academic-style reports

## Workflow

1. Identify the user's immediate task.
2. Load only the relevant reference file from this folder.
3. Follow that file's procedure closely.
4. If the task spans multiple stages, complete the current stage first and then propose the next one.

## Reference Routing

- For abstract writing: read `abstract.md`
- For literature search: read `lit-search.md`
- For citation verification: read `cite-verify.md`
- For citation network analysis: read `citation-network.md`
- For journal matching: read `journal-match.md`
- For research gap analysis: read `research-gap.md`
- For peer review: read `peer-review.md`
- For report design or HTML report generation: read `report-template.md`

## Notes

- Keep the entry skill concise; do not load all reference files unless needed.
- Prefer direct execution for simple drafting tasks.
- Use public web sources only when the selected reference explicitly requires current literature or citation checks.
