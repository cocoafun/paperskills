# PaperSkills Architecture Draft

PaperSkills is moving from a small skill collection to a workflow system for AI-native research.

## Workflow

The target flow is:

`idea -> scope -> track -> synthesize -> design -> draft -> review -> revise`

## Layers

- Entry layer: `using-paperskills`
- Orchestration layer: `research-scoping`, `research-design`, `paper-drafting`, `revision-planning`
- Execution layer: `paper-tracker`, `literature-review`, `peer-review`

## Shared contracts

The repository now includes shared brief schemas in `schemas/` so upstream outputs can be reused downstream instead of passing only free-form text.
