# PaperSkills

PaperSkills is an academic skill library designed for AI agents and human operators.

## What you get

- A multilingual site shell built with Next.js.
- A documentation layer powered by Fumadocs.
- Repository-owned docs content stored inside the submodule.
- A lightweight Drizzle-backed authentication data layer.

## Current skills

- `literature-review`
- `paper-tracker`
- `peer-review`
