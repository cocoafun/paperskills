# PaperSkills

PaperSkills is a research workflow system for AI agents.

It is designed to route a request to the right research stage before the agent starts generating content.

## Workflow overview

`idea -> scope -> track -> synthesize -> design -> draft -> review -> revise`

## Layers

- Entry layer
- Orchestration layer
- Execution layer
