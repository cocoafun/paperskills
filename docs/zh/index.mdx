# PaperSkills

PaperSkills 是一个面向 AI Agent 的科研工作流系统，而不只是一个学术技能仓库。

它的目标是把模糊研究想法转成一条可执行的链路：

`选题收敛 -> 论文追踪 -> 文献综述 -> 研究设计 -> 论文写作 -> 审稿模拟 -> 修订计划`

## 工作流结构

- 入口层：`using-paperskills`
- 编排层：`research-scoping`、`research-design`、`paper-drafting`、`revision-planning`
- 执行层：`paper-tracker`、`literature-review`、`peer-review`

## 你可以在这里获得什么

- 面向科研阶段的 skill 路由，而不是零散 prompt。
- 跨 skill 共享的 brief/schema 结构。
- 面向 Agent 的证据边界与输出契约。
- 中英文文档入口与后续可扩展的贡献规范。
