# PaperSkills

PaperSkills 是一套面向 AI Agent 的学术技能集合，用来帮助你完成选题收敛、文献检索、研究缺口分析、模拟审稿、引文核验与投稿匹配等常见科研任务。

## 你可以获得什么

- 面向 Claude Code、Codex、Cursor 等环境的安装与上手说明。
- 当前核心 skill 的使用文档。
- 按任务类型组织的能力说明和选择建议。

## 当前文档结构

1. `快速开始`
   帮你快速理解 PaperSkills 的使用方式。
2. `安装方式`
   说明如何在 Claude Code、Codex、Cursor 等环境里安装并调用 PaperSkills。
3. `技能介绍`
   统一归档当前核心 skill 的说明与使用方式。
4. 各具体 skill 页面
   分别介绍各个子 skill 的输入、输出、典型用法与注意事项。

## 当前技能

- `topic-framing`
- `lit-search`
- `abstract`
- `cite-verify`
- `citation-network`
- `research-gap`
- `paper-tracker`
- `peer-review`
- `journal-match`

这些 skill 都来自当前仓库中的实际能力定义。

## 推荐阅读顺序

1. 先读 `快速开始`
2. 再看 `安装方式`
3. 按任务类型进入 `技能介绍`
4. 最后阅读对应 skill 的专页
