# 快速开始

如果你第一次接触 PaperSkills，建议先不要急着直接调用某个 skill。更高效的方式是先理解它能解决什么问题、你当前处于哪个任务阶段，再进入对应 skill。

## 推荐上手顺序

1. 先阅读 `安装方式`
2. 再进入 `技能介绍`
3. 根据自己的任务选择合适的 skill
4. 需要时再深入到具体 skill 页面

## 怎么理解这套技能

可以把当前能力粗分为几类任务：

- 想收敛研究题目：用 `/topic-framing`
- 想做一次性文献检索：用 `/lit-search`
- 想找最近的新论文：用 `/paper-tracker`
- 想识别研究缺口：用 `/research-gap`
- 想对论文做模拟审稿：用 `/peer-review`
- 想核查引用：用 `/cite-verify`

## 一个简单工作流

如果你是第一次使用，可以先按下面顺序理解：

1. 用 `paper-tracker` 发现近期相关论文
2. 用 `lit-search` 或 `citation-network` 补齐文献脉络
3. 用 `research-gap` 判断是否存在可做的新问题
4. 在写出草稿后用 `peer-review` 做一轮模拟评审
5. 投稿前再用 `cite-verify` 做引用核验
